# PART 11: VERIFICATION FRAMEWORK

## 11.1 62 Drift Indicators (Condensed by Category)

**Factual Drift (1-15):** Unsourced claim, single-source reliance, outdated info (>90 days), circular citation, authority score <2.0, broken chain to primary, unbounded confidence, gap undocumented, consensus not measured, independence not proven (<3 unique primaries), audit trail missing, unchallenged premise, counter-thesis missing, catalyst age unscored, injection vulnerability.

**Analytical Drift (16-30):** Model selection unjustified, parameter tuning undocumented, correlation without causation, confirmation bias, survivorship bias, recency bias, anchoring, narrative fallacy, availability heuristic, base rate neglect, Simpson's paradox, Goodhart's Law violation, regression to mean ignored, endogeneity, omitted variable.

**Operational Drift (31-45):** Protocol deviation, gate bypass, principal override without log, undocumented change, version mismatch, dead link in protocol file, stale carry-forward (>48hr), credential reference broken, zombie node active, alwaysOutputData ON, hardcoded API key, wrong data flow reference ($json vs $('node').first()), truncated code paste, empty file pushed, SHA mismatch on push.

**Behavioral Drift (46-62):** Placation, hedging without data, apologizing for facts, thesis confirmation over counter-thesis, single-agent output without concurrence, recommendation without confidence score, missing [INFERENCE] labels, "I think" without attribution, fabricated attribution, scope creep, task looping (repeating same ask), promise without delivery proof, session close without GitHub push, carry-forward without restart prompt, memory claim without search verification.
