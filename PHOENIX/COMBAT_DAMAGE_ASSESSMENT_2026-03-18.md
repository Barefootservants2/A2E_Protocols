## ROOT CAUSE ANALYSIS

### 1. IRONCLAD's Stop Framework Is Wrong for Thesis Positions
IRONCLAD v2.0 applies stops uniformly across all position types. This is a fundamental 
design error. Stops are appropriate for tactical/swing trades where you have a defined 
entry thesis and a defined exit point. They are COUNTERPRODUCTIVE for structural thesis 
positions where the thesis is multi-year and volatility is the weapon used against you.

**Fix:** IRONCLAD v3.0 must distinguish:
- Ring 1 (SGOV/JEPI): No stops needed — income/safety
- Ring 2 (PSLV/IBIT/physical): NO STOPS — thesis holds, alerts only
- Ring 3 (AG/SIL/miners): NO STOPS — thesis holds, alerts only
- Ring 4 (tactical swings): STOPS APPROPRIATE — defined risk
- Ring 5 (lottery): STOPS APPROPRIATE — defined risk

### 2. Automation Gap Is an Emergency
Nothing is automated. Everything runs on manual checks in conversational AI sessions 
that reset every time. The defense systems exist as documents, not as running code.

**Critical automation needed (in priority order):**
1. COMEX inventory daily pull (H42) — protects thesis conviction
2. EDGAR 13F polling weekly (GAP1) — shows institutional positioning
3. COT report weekly pull — shows managed money positioning
4. Price alert system via n8n/Telegram — replaces E*TRADE stop mechanism
5. GABRIEL overnight watch — catches overnight events

### 3. Session Model Cannot Deliver Engineering
Discovery-driven conversational sessions produce research and specs, not deployed systems. 
The carry-forward model loses context and priority between sessions. Engineering delivery 
requires a sprint model with defined deliverables per session.

**Fix:** Adopt a strict sprint structure:
- Session starts: read carry-forward, confirm objective, NO PIVOTS
- Session body: build ONE thing to production
- Session end: deploy, verify, document, carry-forward
- No research sessions until automation backlog is cleared

### 4. Same-Day Re-Entry Violates IRONCLAD
IRONCLAD explicitly bans same-day re-entry. It was violated on 3/16 (re-entered SIL, 
HYMC, AG same day as stop-out) and on 3/10 (re-entered COPX, SIL, PSLV, SLVR same day).
