# Advanced Prompt Chains

A single prompt produces a snapshot. A chain produces an investigation. This chapter contains multi-step sequences where each prompt builds on the output of the previous one, creating layered analysis that no single query can achieve.

Each chain includes:

- **Chain purpose** — what the sequence produces when completed
- **Number of steps** — typically 4-8 prompts in sequence
- **Estimated completion time** — wall-clock time for the full chain
- **Gate dependencies** — which AIORA gates the chain addresses
- **The complete prompt sequence** — with carry-forward instructions between steps
- **Decision point markers** — where human judgment is required before proceeding
